Introduction

Base URL: http://LYT-SaaS-BED-Portfolio.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Request      

POST api/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

nickname   string   

Must not be greater than 255 characters. Example: txtazkggcclttb

given_name   string   

Must not be greater than 255 characters. Example: tffsepzkvkmgyb

family_name   string   

Must not be greater than 255 characters. Example: emshzbweurspjyhki

user_type   string   

Example: applicant

Must be one of:
  • client
  • staff
  • applicant
status   string   

Example: unknown

Must be one of:
  • active
  • unconfirmed
  • suspended
  • banned
  • unknown
email   string   

Must be a valid email address. Example: alize63@example.net

password   string   

Example: aut
