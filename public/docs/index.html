Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Request      

POST api/positions

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

advertising_start_date   string   

Must be a valid date in the format Y-m-d. Example: 2024-09-12

advertising_end_date   string   

Must be a valid date in the format Y-m-d. Example: 2024-09-12

position_title   string   

Must not be greater than 255 characters. Example: biiqttrbcziamm

position_description   string   

Must not be greater than 500 characters. Example: shq

position_keywords   string  optional  

Must not be greater than 255 characters. Example: hqtqvlpb

minimum_salary   number   

Must be at least 0. Example: 72

maximum_salary   number   

Must be at least 0. Example: 81

salary_currency   string   

Must be 3 characters. Example: gnw

company_id   string   

Example: quia

benefits   string  optional  

Must not be greater than 500 characters. Example: rxgxpykbnfkwvnemxvqiiiv

requirements   string  optional  

Must not be greater than 500 characters. Example: relqacxcplajqdrzmatvocmq

position_type   string   

Example: permanent

Must be one of:
  • permanent
  • contract
  • part-time
  • casual
  • internship
